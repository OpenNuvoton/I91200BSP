---------------------------------------------------------------------------------------------------------
Purpose:
---------------------------------------------------------------------------------------------------------
    Demonstrate Aanlog or Digital Mic input to SDADC
---------------------------------------------------------------------------------------------------------
Operation:
---------------------------------------------------------------------------------------------------------
  I91200BS version :   
    (1) used ISD_DEMO91200_EVB_Bridge_Sensor V1.3 board.
    (2) connect your DMIC, pin B14(clock), pin B12(Data),pin VDD, pin GND, pin L/R  
    (2) Compiled to execute
    (3) Program test procedure -
        1.press '0' to start recording .
  I91200 version :
    (1) Compiled to execute
    (2) Program test procedure -
        1.press '0' to start recording , but you must use ISD_DEMO91200_EVB_Bridge_Sensor V1.3 board.
        2.press '1' to start recording , you must use ISD_DEMO91260_EVB V1.0, AMIC on board.
