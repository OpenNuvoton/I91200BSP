---------------------------------------------------------------------------------------------------------
Purpose:
---------------------------------------------------------------------------------------------------------
  Demonstrate  the usage of gpio driver, includes
   (1) Output mode.
   (2) Input mode
   (3) External interrupt(trigger irq when high to low(falling))
    
---------------------------------------------------------------------------------------------------------
Operation:
---------------------------------------------------------------------------------------------------------
    (1) Compiled to execute
    (2) Program test procedure -
        1. Config output low to light LED.
        2. Press any key on keyboard to next demo
        3. Press SWB0 on EVB to next demo
        4. Press SWB2 on EVB to trigger irq
        5. End demo gpio sample when gpio irq has been trigger.
