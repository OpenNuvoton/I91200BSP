---------------------------------------------------------------------------------------------------------
Purpose:
---------------------------------------------------------------------------------------------------------
    Demonstrate how to use analog Mic input to SDADC and using ALC to output.
    (1)	Using CRC-CCITT in this demo sample.
    (2)	Calculate the checksum of a set of constant data.
		
---------------------------------------------------------------------------------------------------------
Operation:
---------------------------------------------------------------------------------------------------------
    (1) Connects to com port to send out demo message (TX=GPA4, RX=GPA5).
    (2) Connect speaker to target board.
    (3) Compiled to execute.
    (4) Program test procedure -
        1. After showing "Press SWB5 to toggle ALC." and "Press SWB4 to toggle Volume Control."
	2. You can give Mic the sound.
	3. button SWB5 for voice more louder. 
	4. button SWB4 for voice Smaller
